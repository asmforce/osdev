1. Маскировка прерываний:
Тут мы задаем маску, что позволяет маскировать выборочные IRQ

  wrio8( 0xFF, 0x21 ); // 0x21 - в ведущий контроллер прерываний
  wrio8( 0xFF, 0xA1 ); // 0xA1 - в ведомый контроллер прерываний

Порты 0x21 0xA1 доступны как для записи, так и для чтения, но
при маскировке отдельных прерываний нужно сначала считать маску,
изменить ее, а затем записать. В таком случае, нужно вставлять
задержку между чтением и записью, т.к. контроллер i8259 или
более новый совместимый может не обеспечить такую скорость

2. EOI - End of interrupt:
Отправляем команду завершения обработки прерывания, подтвердив,
что прерывание обработано.

  wrio8( 0x20, 0x20 ); // 0x20 - в ведущий контроллер прерываний
  wrio8( 0x20, 0xA0 ); // 0xA0 - в ведомый контроллер прерываний
